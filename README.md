# Unit Conversion API

A RESTful ASP.NET Core Web API for converting units of measurement.

## Features

- Length Conversion
  - Meter ↔ Feet
  - Kilometer ↔ Mile

- Weight Conversion
  - Kilogram ↔ Pound

- Temperature Conversion
  - Celsius ↔ Fahrenheit

- Swagger UI Support
- Dependency Injection
- Clean Architecture

## Technologies Used

- ASP.NET Core Web API
- C#
- Swagger/OpenAPI

## Running the Project

1. Clone the repository
2. Open terminal in project folder
3. Run:

```bash
dotnet restore
dotnet run
